Recent Projects

Multicraft

I have been working on this project with guidance from Professor Marcelo Worsley at tiilt. In this project, I am designing and actively developing multimodal interfaces such as Gaze (Tobii Eyetracking with C#) and Speech (currently IBM Watson AI Speech to Text with Python) to empower individuals with physical impairments to play and create in Minecraft. Both interfaces interact with Minecraft through a locally-hosted server (Spigot server using Bukkit server plugins with Java). You can see the current repository here.

Time Series Analysis of Environmental Data

I worked on this project throughout my internship at Argonne National Laboratory as part of a research initiative led by Pete Beckman. The project involved designing a new PostgreSQL database extended with TimescaleDB to manage the time-series environmental data and metadata gathered by nodes maintained and used by the Array of Things research group. Once in use, the time-series data from the database was then used for developing a temperature sensor calibration model utilizing a combination of unsupervised and supervised machine learning models using Python and scikit-learn. The new model lowered reported temperature error for nodes in the Chicago area during the summer by about 60%.

DuraCot

This project was a part of Design Thinking and Communication (DTC), a course in Northwestern's Engineering First Program in the McCormick School of Engineering. Working in a small team of students, we designed and developed a collapsible changing table to facilitate diaper changing of a 9-year old with cerebral palsy. The portable changing table allows the caretaker to change the diaper within the privacy of a handicap stall of most public restrooms. This project involved solving real-world problems and the concept of human-centered design.
